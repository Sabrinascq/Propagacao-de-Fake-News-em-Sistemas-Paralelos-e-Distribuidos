# Propaga-o-de-Fake-News-em-Sistemas-Paralelos-e-Distribu-dos
O trabalho consiste na transformação e evolução de uma aplicação sequencial para versões paralela e distribuída.  O problema aborda a simulação da propagação de fake news em uma população representada por uma matriz bidimensional, inspirada em modelos de autômatos celulares e dinâmica de propagação de informação.
